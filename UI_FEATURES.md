# Интерфейс многоагентной системы

## Новые возможности

### 1. Панель статуса агентов (TreeView)

**Расположение**: Боковая панель → CursorAI → Агенты

Отображает:
- Всех агентов системы
- Статус каждого агента (Работает, Ожидает, Ошибка, Отключен)
- Текущие задачи агентов
- Статистику (задач в работе, выполнено)

**Команды**:
- Клик по агенту → раскрыть список задач
- Клик по задаче → просмотр деталей

### 2. WebView панель статуса

**Открытие**: 
- Команда: `Cursor Autonomous: Show Status Panel`
- Горячая клавиша: `Ctrl+Shift+S` (Windows/Linux) или `Cmd+Shift+S` (macOS)
- Кнопка в статус-баре: `$(dashboard) Status`

**Возможности**:
- Общая статистика системы
- Карточки всех агентов с детальной информацией
- Текущие задачи каждого агента
- Автообновление каждые 5 секунд
- Кнопка ручного обновления

### 3. Улучшенный статус-бар

**Кнопки в статус-баре**:
- `$(robot) CursorAI` - Быстрое меню (Ctrl+Shift+A)
- `$(person) Virtual User` - Переключение виртуального пользователя (Ctrl+Shift+V)
- `$(search) Analyze` - Анализ проекта (Ctrl+Shift+P)
- `$(dashboard) Status` - Панель статуса (Ctrl+Shift+S)

**Статус индикаторы**:
- `$(check)` - Система активна
- `$(circle-slash)` - Система остановлена
- Цвет фона меняется в зависимости от статуса

### 4. Быстрое меню

**Открытие**: `Ctrl+Shift+A` или клик по иконке в статус-баре

**Опции**:
- Запустить/остановить оркестратор
- Включить/выключить виртуального пользователя
- Анализ проекта
- Показать статус системы
- Открыть настройки

### 5. Детали агента

**Просмотр деталей**:
- Команда: `Cursor Autonomous: Show Agent Details`
- Клик по агенту в TreeView
- Клик по агенту в WebView панели

**Информация**:
- Название и статус агента
- Количество задач в работе
- Количество выполненных задач
- Текущая задача (если есть)
- Список всех задач агента

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `Ctrl+Shift+A` | Быстрое меню |
| `Ctrl+Shift+V` | Переключить виртуального пользователя |
| `Ctrl+Shift+S` | Панель статуса |
| `Ctrl+Shift+P` | Анализ проекта |

## Автоматическое обновление

Система автоматически обновляет статус:
- **TreeView**: каждые 3 секунды
- **WebView панель**: каждые 5 секунд
- **Статус-бар**: каждые 5 секунд

## Агенты системы

1. **Оркестратор** - координатор работы всех агентов
2. **Backend Developer** - разработка backend
3. **Frontend Developer** - разработка frontend
4. **Software Architect** - архитектура и планирование
5. **Data Analyst** - анализ данных и производительности
6. **DevOps Engineer** - инфраструктура и деплой
7. **QA Engineer** - тестирование
8. **Виртуальный пользователь** - автономное принятие решений
9. **Система самосовершенствования** - улучшение правил и агентов

## Статусы агентов

- **Работает** - агент выполняет задачу
- **Ожидает** - агент готов к работе
- **Ошибка** - произошла ошибка
- **Отключен** - агент отключен

## Задачи

Каждая задача имеет:
- **Тип**: feature, bug, improvement, refactoring, documentation
- **Приоритет**: high, medium, low
- **Статус**: pending, in-progress, completed, blocked
- **Назначенный агент**: какой агент работает над задачей

## Использование

### Просмотр статуса

1. Откройте боковую панель CursorAI
2. Выберите "Агенты"
3. Раскройте агента для просмотра его задач

### Детальная панель

1. Нажмите `Ctrl+Shift+S` или кликните на кнопку Status в статус-баре
2. Просмотрите общую статистику
3. Кликните на агента для просмотра деталей

### Управление задачами

Задачи создаются автоматически:
- При анализе проекта
- Виртуальным пользователем
- Системой самосовершенствования

Статус задач обновляется автоматически при работе агентов.
