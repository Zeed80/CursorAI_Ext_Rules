# Быстрая установка - Перетаскивание .vsix файла

## Самый простой способ установки!

### Шаг 1: Соберите расширение

Если у вас уже есть собранный `.vsix` файл, переходите к шагу 2.

Если нет, соберите расширение одним из способов:

**Автоматически:**
```bash
npm run build
```

**Или вручную:**
```bash
npm install
npm run compile
npm run package
```

После сборки вы найдете файл `cursor-ai-autonomous-extension-0.1.0.vsix` в корне проекта.

### Шаг 2: Установка перетаскиванием

1. **Откройте CursorAI**

2. **Откройте панель расширений:**
   - Нажмите `Ctrl+Shift+X` (Windows/Linux)
   - Или `Cmd+Shift+X` (macOS)
   - Или кликните на иконку расширений в боковой панели

3. **Перетащите .vsix файл:**
   - Найдите файл `cursor-ai-autonomous-extension-0.1.0.vsix` в проводнике/файловом менеджере
   - Перетащите его в окно CursorAI (в область панели расширений)
   - CursorAI автоматически установит расширение

4. **Готово!**
   - Перезапустите CursorAI (если потребуется)
   - Расширение активируется автоматически при открытии проекта

## Альтернативный способ: Install from VSIX

Если перетаскивание не работает:

1. Откройте панель расширений (`Ctrl+Shift+X`)
2. Нажмите на **три точки (`...`)** вверху панели
3. Выберите **"Install from VSIX..."**
4. Выберите файл `cursor-ai-autonomous-extension-0.1.0.vsix`

## Проверка установки

1. Откройте командную палитру: `Ctrl+Shift+P` (или `Cmd+Shift+P`)
2. Введите "Cursor Autonomous"
3. Вы должны увидеть команды:
   - `Cursor Autonomous: Start Orchestrator`
   - `Cursor Autonomous: Enable Virtual User Mode`
   - И другие команды

## Готовые .vsix файлы

Если вы хотите поделиться расширением с другими:

1. Соберите расширение: `npm run build`
2. Файл `.vsix` можно передать другим пользователям
3. Они могут установить его перетаскиванием или через "Install from VSIX"

**Важно:** Файл `.vsix` уже является "одним файлом" - это архив, содержащий все необходимое для расширения.
